🪄 변환 영역

이미지/텍스트/PDF/오피스/HWP 파일을 PDF로 변환합니다.

📥 파일을 여기로 드래그하거나 아래 버튼으로 추가

    🧩 병합 영역

    PDF 파일을 업로드 순서대로 정렬 후 병합합니다.

    📎 PDF를 여기로 드래그앤드롭

    파일을 추가해 작업을 시작하세요.

    ✨ 이 도구로 할 수 있는 작업

    • PNG, JPG, WEBP 등 이미지를 A4 비율 PDF로 변환
    • TXT, CSV, JSON, MD 같은 텍스트 파일을 PDF 문서로 변환
    • DOC, DOCX, XLS, XLSX, PPT, PPTX, HWP 파일을 PDF로 변환
    • 변환 완료 파일을 병합 영역으로 즉시 이동
    • PDF 아이콘을 드래그해 순서를 바꾼 뒤 원하는 순서로 병합

    🧭 작업 가이드

    1. 왼쪽 변환 영역에 파일을 드래그앤드롭하거나 파일 추가 버튼으로 업로드합니다.
    2. 전체 PDF 변환 버튼을 눌러 파일별 변환 결과를 확인합니다.
    3. 중앙의 변환 완료 -> 병합 버튼으로 결과를 오른쪽으로 이동합니다.
    4. 오른쪽 병합 영역에서 아이콘 순서를 원하는 대로 바꿉니다.
    5. 순서대로 병합 PDF 생성을 눌러 최종 파일을 다운로드합니다.

    ❓ 자주 묻는 질문

    Q. 업로드한 파일은 서버에 저장되나요?

    현재 버전은 브라우저 내에서만 파일을 처리하도록 설계되어, 일반적인 사용 과정에서 업로드 파일이 별도 서버에 저장되지 않습니다.

    Q. 병합 순서는 어떻게 결정되나요?

    병합 영역에 표시된 아이콘의 번호 순서대로 병합됩니다. 아이콘을 드래그해 순서를 변경하면 번호가 즉시 다시 계산됩니다.

    Q. 모든 형식이 변환되나요?

    이미지, 텍스트, PDF, DOC/DOCX/XLS/XLSX/PPT/PPTX/HWP 형식을 지원합니다. 서버 환경에 따라 일부 문서는 폰트/레이아웃이 다를 수 있습니다.
